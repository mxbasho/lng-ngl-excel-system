# Project Documentation

## Overview
This project aims to simplify the management and utilization of Excel files through a dedicated API. The main features include:

- Uploading Excel files
- Parsing and extracting data from Excel sheets
- Performing data operations (filter, sort, etc.)
- Downloading modified Excel files

## Setup Instructions

### Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
- Clone the repository:

  ```bash
  git clone https://github.com/mxbasho/lng-ngl-excel-system.git
  cd lng-ngl-excel-system
  ```

### Installation
Run the following command to install the project dependencies:

```bash
npm install
```

### Usage
To start the project, use the following command:

```bash
npm start
```

### API Documentation
Refer to the `API.md` file included in the repository for detailed information about the available endpoints and their usage.
